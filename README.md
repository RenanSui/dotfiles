# Hyprland Dotfiles

Theme based on anime Bocchi The Rock

## Installation

Clone the repository:
```bash
  git clone https://github.com/RenanSui/dotfiles.git ~/dotfiles
```

Navigate to the dotfiles directory:
```bash
  cd ~/dotfiles
```

Install Dependecies
```bash
pacman -S hyprland kitty wofi cliphist fastfetch waybar mako grim slurp networkmanager network-manager-applet polkit polkit-kde-agent vlc waybar bluez gtk4 gtk4-layer-shell pipewire helvum pamixer pwvucontrol ttf-jetbrains-mono-nerd gnome-keyring libsecret yazi
```

yay -S orbit-wifi
